Instalação: Instalar node, npm e nestjs
$ npm install -g npm
$ npm install -g @nestjs/cli

// Criar projeto nestjs
$ nest new nome-projeto

// Rodar aplicação nest
$ npm run start:dev

// Criar module
$ nest g module jogadores

// Criar controller
$ nest g controller jogadores
